# IO 组件方案

## 1. 组件概述

`opencode-base-io` 是基于 Java NIO.2 API 的核心 IO 工具集，提供文件/目录操作、资源加载、流处理、校验和计算、批量操作、MIME 类型检测等高频 IO 操作的便捷封装。所有方法使用非受检异常（`OpenIOOperationException`），简化调用方代码。零外部依赖，支持 JDK 25 虚拟线程。

## 3. 核心 API

### 3.1 OpenIO

> 文件和目录操作的主门面类，基于 NIO.2 Path/Files，提供读写、复制、移动、删除、遍历等操作。所有 IO 异常转为非受检异常。

**主要方法:**

| 方法 | 描述 |
|------|------|
| `readBytes(Path path)` | 读取文件为字节数组 |
| `readString(Path path)` | 以 UTF-8 读取文件为字符串 |
| `readString(Path path, Charset charset)` | 以指定编码读取文件为字符串 |
| `readLines(Path path)` | 以 UTF-8 读取文件所有行 |
| `readLines(Path path, Charset charset)` | 以指定编码读取文件所有行 |
| `lines(Path path)` | 返回延迟加载的行流 |
| `readFirstLine(Path path)` | 读取文件第一行 |
| `writeBytes(Path path, byte[] bytes, OpenOption... options)` | 写入字节数组到文件 |
| `writeString(Path path, CharSequence content, OpenOption... options)` | 以 UTF-8 写入字符串到文件 |
| `writeString(Path path, CharSequence content, Charset charset, OpenOption... options)` | 以指定编码写入字符串 |
| `writeLines(Path path, Iterable<? extends CharSequence> lines, OpenOption... options)` | 写入多行文本 |
| `append(Path path, CharSequence content)` | 追加内容到文件 |
| `appendLines(Path path, Iterable<? extends CharSequence> lines)` | 追加多行到文件 |
| `exists(Path path, LinkOption... options)` | 检查路径是否存在 |
| `isFile(Path path)` | 检查是否为文件 |
| `isDirectory(Path path)` | 检查是否为目录 |
| `isEmptyDirectory(Path path)` | 检查是否为空目录 |
| `isSymbolicLink(Path path)` | 检查是否为符号链接 |
| `isHidden(Path path)` | 检查是否为隐藏文件 |
| `isReadable(Path path)` | 检查是否可读 |
| `isWritable(Path path)` | 检查是否可写 |
| `isSameFile(Path path1, Path path2)` | 检查是否为同一文件 |
| `createFile(Path path)` | 创建文件 |
| `createDirectory(Path path)` | 创建目录 |
| `createDirectories(Path path)` | 递归创建目录 |
| `delete(Path path)` | 删除文件或空目录 |
| `deleteIfExists(Path path)` | 若存在则删除 |
| `deleteRecursively(Path path)` | 递归删除目录及内容 |
| `copy(Path source, Path target, CopyOption... options)` | 复制文件 |
| `copyRecursively(Path source, Path target, CopyOption... options)` | 递归复制目录 |
| `move(Path source, Path target, CopyOption... options)` | 移动/重命名文件 |
| `size(Path path)` | 获取文件大小 |
| `directorySize(Path path)` | 获取目录总大小 |
| `getLastModifiedTime(Path path)` | 获取最后修改时间 |
| `setLastModifiedTime(Path path, Instant time)` | 设置最后修改时间 |
| `list(Path dir)` | 列出目录内容 |
| `walk(Path start)` | 递归遍历目录树 |
| `walk(Path start, int maxDepth)` | 遍历目录树（限制深度） |
| `glob(Path start, String pattern)` | 按 glob 模式搜索文件 |
| `find(Path start, String regex)` | 按正则表达式搜索文件 |
| `find(Path start, int maxDepth, BiPredicate<Path, BasicFileAttributes> matcher)` | 自定义条件搜索文件 |

**示例:**

```java
// 读取文件
String content = OpenIO.readString(path);
List<String> lines = OpenIO.readLines(path);
byte[] bytes = OpenIO.readBytes(path);

// 写入文件
OpenIO.writeString(path, "Hello World");
OpenIO.writeLines(path, List.of("line1", "line2"));
OpenIO.append(path, "\nNew content");

// 文件操作
OpenIO.copy(source, target);
OpenIO.move(source, target);
OpenIO.deleteRecursively(dir);

// 目录遍历
try (Stream<Path> files = OpenIO.glob(dir, "*.java")) {
    files.forEach(System.out::println);
}
```

### 3.6 OpenFileVisitors

> 文件遍历访问器工具类，提供常用 FileVisitor 实现。

**主要方法:**

| 方法 | 描述 |
|------|------|
| `deleteVisitor()` | 创建递归删除访问器 |
| `copyVisitor(Path source, Path target, CopyOption... options)` | 创建递归复制访问器 |
| `actionVisitor(Consumer<Path> action)` | 创建自定义操作访问器 |

**内部类:**

| 类 | 描述 |
|------|------|
| `CountingVisitor` | 统计文件数（`getFileCount()`）、目录数（`getDirectoryCount()`）、总数（`getTotalCount()`） |
| `SizeVisitor` | 统计总大小（`getTotalSize()`） |

**示例:**

```java
// 递归删除目录
Files.walkFileTree(dir, OpenFileVisitors.deleteVisitor());

// 递归复制目录
Files.walkFileTree(source, OpenFileVisitors.copyVisitor(source, target));

// 自定义操作
Files.walkFileTree(dir, OpenFileVisitors.actionVisitor(path -> {
    System.out.println(path);
}));

// 统计文件数
var counter = OpenFileVisitors.countingVisitor();
Files.walkFileTree(dir, counter);
System.out.println("Files: " + counter.getFileCount());
```

### 3.31 TeeInputStream

> 分流输入流，读取时同步写入到分支输出流，类似 Unix tee 命令。

| `read(byte[] b, int off, int len)` | 读取并分流 |
| `getBranch()` | 获取分支输出流 |

**示例:**

```java

}
```
