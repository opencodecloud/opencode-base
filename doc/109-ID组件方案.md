# ID 组件方案

## 1. 组件概述

`opencode-base-id` 是一个全面、高性能的分布式 ID 生成器工具库，提供 Snowflake、UUID v4/v7、ULID、TSID、KSUID、NanoID 和号段模式七种主流算法。通过统一的 `IdGenerator<T>` 接口和 `OpenId` 门面类提供简洁 API，同时内置 ID 解析、格式转换、时钟回拨处理、WorkerId 自动分配等企业级功能，基于 JDK 25 实现，零外部依赖。

**适用场景：**
- 分布式系统唯一标识生成（订单号、用户 ID、流水号）
- 时间有序 ID（日志追踪、事件排序）
- 短链接/短 ID（Base62 编码）
- 号段模式批量预取（高并发场景）

**模块依赖：**
```xml
<dependency>
    <groupId>cloud.opencode.base</groupId>
    <artifactId>opencode-base-id</artifactId>
    <version>${version}</version>
</dependency>
```

## 3. 核心 API

### 3.1 OpenId

> ID 生成门面入口类，提供所有 ID 算法的快捷访问。内部维护默认的 Snowflake、UUID v7、ULID、TSID、KSUID 生成器单例。

| `static String snowflakeIdStr()` | 生成 Snowflake ID（String） |
| `static IdGenerator<Long> getSnowflake()` | 获取默认 Snowflake 生成器 |
| `static IdGenerator<Long> createSnowflake(long workerId, long datacenterId)` | 创建自定义 Snowflake |
| `static SnowflakeBuilder snowflakeBuilder()` | 获取 Snowflake Builder |
| `static SnowflakeIdParser.ParsedId parseSnowflakeId(long id)` | 解析 Snowflake ID |
| `static UUID uuid()` | 生成 UUID v4 |
| `static String uuidStr()` | 生成 UUID v4 字符串 |
| `static String simpleUuid()` | 生成无横线 UUID |
| `static UUID uuidV7()` | 生成 UUID v7（时间有序） |
| `static String uuidV7Str()` | 生成 UUID v7 字符串 |
| `static IdGenerator<UUID> getUuidV7Generator()` | 获取 UUID v7 生成器 |
| `static String ulid()` | 生成 ULID 字符串 |
| `static byte[] ulidBytes()` | 生成 ULID 字节数组 |
| `static IdGenerator<String> getUlidGenerator()` | 获取 ULID 生成器 |
| `static UlidParser.ParsedUlid parseUlid(String ulid)` | 解析 ULID |
| `static long tsid()` | 生成 TSID（long） |
| `static String tsidStr()` | 生成 TSID 字符串（Crockford Base32） |
| `static IdGenerator<Long> getTsidGenerator()` | 获取 TSID 生成器 |
| `static TsidGenerator createTsidGenerator(int nodeBits, long nodeId)` | 创建自定义 TSID |
| `static TsidParser.ParsedTsid parseTsid(String tsidStr)` | 解析 TSID 字符串 |

| `static IdGenerator<Long> createSegmentGenerator(SegmentAllocator allocator, String bizTag)` | 创建指定业务标签的号段生成器 |

### 3.5 SnowflakeGenerator

> Snowflake 分布式 ID 生成器，默认 41 位时间戳 + 5 位数据中心 + 5 位机器 + 12 位序列号，支持自定义位分配。

**主要方法：**

| 方法 | 描述 |
|------|------|
| `static SnowflakeGenerator create()` | 使用自动分配的 workerId/datacenterId |
| `static SnowflakeGenerator create(long workerId, long datacenterId)` | 指定 workerId/datacenterId |
| `static SnowflakeBuilder builder()` | 获取 Builder |
| `Long generate()` | 生成 ID |
| `String generateStr()` | 生成 ID 字符串 |
| `long getWorkerId()` | 获取 workerId |
| `long getDatacenterId()` | 获取 datacenterId |
| `long getEpoch()` | 获取起始纪元 |
| `SnowflakeConfig getConfig()` | 获取配置 |
| `SnowflakeIdParser.ParsedId parse(long id)` | 解析 ID |
| `String getType()` | 返回 "snowflake" |

**示例：**

```java
// 快速创建
SnowflakeGenerator gen = SnowflakeGenerator.create();
long id = gen.generate();

// 指定节点
SnowflakeGenerator gen2 = SnowflakeGenerator.create(1, 1);

// Builder 模式
SnowflakeGenerator gen3 = SnowflakeGenerator.builder()
    .workerId(1)
    .datacenterId(1)
    .epoch(Instant.parse("2024-01-01T00:00:00Z"))
    .clockBackwardStrategy(Wait.ofSeconds(5))
    .sequenceBits(12)
    .build();

// 解析
SnowflakeIdParser.ParsedId parsed = gen.parse(id);
System.out.println("时间: " + parsed.time());
System.out.println("序列号: " + parsed.sequence());
```

### 3.9 ClockBackwardStrategy（sealed）

> 时钟回拨处理策略密封接口，三种实现：`Wait`、`ThrowException`、`Extend`。

```java
public sealed interface ClockBackwardStrategy
    permits Wait, ThrowException, Extend
```

**3.9.1 Wait**

> 等待策略，等待时钟追上最后时间戳。超过最大等待时间则抛异常。

| 方法 | 描述 |
|------|------|
| `Wait(long maxWaitMillis)` | 指定最大等待毫秒数 |
| `static Wait ofSeconds(int seconds)` | 秒级等待 |
| `static Wait ofMillis(long millis)` | 毫秒级等待 |
| `long maxWaitMillis()` | 获取最大等待时间 |
| `long handle(long lastTimestamp, long currentTimestamp)` | 处理回拨 |

**3.9.2 ThrowException**

> 抛异常策略（单例），直接抛出 `OpenIdGenerationException`。

| 方法 | 描述 |
|------|------|
| `static ThrowException getInstance()` | 获取单例 |
| `long handle(long lastTimestamp, long currentTimestamp)` | 处理回拨（抛异常） |

**3.9.3 Extend**

> 扩展位策略，通过扩展位标记时钟回拨区间。

| 方法 | 描述 |
|------|------|
| `Extend(int extensionBits)` | 指定扩展位数 |
| `int extensionBits()` | 获取扩展位数 |
| `int extensionValue()` | 获取当前扩展值 |
| `int maxExtension()` | 获取最大扩展值 |
| `void reset()` | 重置扩展值 |
| `long handle(long lastTimestamp, long currentTimestamp)` | 处理回拨 |

**示例：**

```java
// 等待 5 秒
ClockBackwardStrategy wait = Wait.ofSeconds(5);

// 直接抛异常
ClockBackwardStrategy throwEx = ThrowException.getInstance();

// 扩展位
ClockBackwardStrategy extend = new Extend(2);
```
