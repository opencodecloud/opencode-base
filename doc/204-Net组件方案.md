# Net 组件方案

## 1. 组件概述

`opencode-base-net` 是基于 JDK 25 HttpClient 的现代化网络工具组件，提供 HTTP 客户端、文件下载、IP 工具、拦截器链、响应式 HTTP、SSE、HTTP/2 Server Push、Mock 服务器等功能。零第三方依赖，完全基于 JDK 原生 API，充分利用 Virtual Threads 和 HTTP/2。

## 3. 核心 API

### 3.1 OpenHttp

> 静态 HTTP 工具门面，内部使用默认 HttpClient，提供最简化的 HTTP 操作方法。

**主要方法:**

| 方法 | 描述 |
|------|------|
| `static String get(String url)` | 发送 GET 请求并返回响应体字符串 |
| `static String get(String url, Map<String, String> headers)` | 带自定义头部的 GET 请求 |
| `static <T> T getAs(String url, Class<T> clazz)` | GET 请求并将响应解析为指定类型 |
| `static byte[] getBytes(String url)` | GET 请求返回字节数组 |
| `static CompletableFuture<HttpResponse> getAsync(String url)` | 异步 GET 请求 |
| `static String postJson(String url, Object body)` | POST JSON 请求 |
| `static <T> T postJson(String url, Object body, Class<T> resultClass)` | POST JSON 并解析响应 |
| `static String postForm(String url, Map<String, String> formData)` | POST 表单请求 |
| `static String post(String url, String body, String contentType)` | 自定义 Content-Type 的 POST |
| `static CompletableFuture<HttpResponse> postJsonAsync(String url, Object body)` | 异步 POST JSON |
| `static String putJson(String url, Object body)` | PUT JSON 请求 |
| `static <T> T putJson(String url, Object body, Class<T> resultClass)` | PUT JSON 并解析响应 |
| `static String patchJson(String url, Object body)` | PATCH JSON 请求 |
| `static String delete(String url)` | DELETE 请求 |
| `static String delete(String url, Map<String, String> headers)` | 带头部的 DELETE 请求 |
| `static Path download(String url, Path savePath)` | 下载文件 |
| `static CompletableFuture<Path> downloadAsync(String url, Path savePath)` | 异步下载文件 |
| `static String uploadFile(String url, Path file, String fileName)` | 上传单个文件 |
| `static String uploadFiles(String url, Map<String, Path> files)` | 上传多个文件 |
| `static HttpClient getClient()` | 获取默认 HttpClient |
| `static void setClient(HttpClient client)` | 设置默认 HttpClient |
| `static void setTimeout(int connectTimeoutMs, int readTimeoutMs)` | 设置超时时间 |
| `static HttpRequest.Builder request()` | 创建请求构建器 |
| `static HttpRequest.Builder request(String url)` | 创建指定 URL 的请求构建器 |
| `static boolean isReachable(String url)` | 检查 URL 是否可达 |
| `static Map<String, String> getHeaders(String url)` | 获取响应头 |

**示例:**

```java
// 简单 GET
String html = OpenHttp.get("https://example.com");

// GET 并解析为对象
User user = OpenHttp.getAs("https://api.example.com/users/1", User.class);

// POST JSON
String result = OpenHttp.postJson("https://api.example.com/users", newUser);

// 下载文件
Path file = OpenHttp.download("https://example.com/file.zip", Path.of("./file.zip"));

// 上传文件
String resp = OpenHttp.uploadFile("https://api.example.com/upload",
    Path.of("/path/to/file.pdf"), "document.pdf");
```

| `HttpResponse postJson(String url, Object body)` | POST JSON |
| `HttpResponse postForm(String url, Map<String, String> formData)` | POST 表单 |
| `HttpResponse putJson(String url, Object body)` | PUT JSON |
| `HttpResponse delete(String url)` | DELETE |
| `HttpResponse patchJson(String url, Object body)` | PATCH JSON |
| `HttpResponse send(HttpRequest request)` | 发送自定义请求 |

| `String getBaseUrl()` | 获取基础 URL |
| `java.net.http.HttpClient getJdkClient()` | 获取底层 JDK HttpClient |

**Builder 方法:**

| 方法 | 描述 |
|------|------|
| `Builder config(HttpConfig config)` | 设置配置 |
| `Builder connectTimeout(int timeoutMs)` | 连接超时(毫秒) |
| `Builder readTimeout(int timeoutMs)` | 读取超时(毫秒) |
| `Builder baseUrl(String baseUrl)` | 设置基础 URL |
| `Builder addInterceptor(Interceptor interceptor)` | 添加拦截器 |
| `Builder defaultHeader(String name, String value)` | 添加默认头部 |

| `Builder http2()` | 启用 HTTP/2 |

**示例:**

```java
// 创建自定义客户端
HttpClient client = HttpClient.builder()
    .baseUrl("https://api.example.com")
    .connectTimeout(5000)
    .readTimeout(30000)
    .addInterceptor(AuthInterceptor.bearer("my-token"))
    .addInterceptor(new LoggingInterceptor())
    .http2()
    .build();

HttpResponse response = client.get("/users/1");
String body = response.body();
```

```java
    .pathParam("id", "123")
    .queryParam("fields", "name,email")
    .sendAndParse(User.class);
```

```java
    .assertSuccessful()
    .body();
```
