# Pdf 组件方案

## 1. 组件概述

`opencode-base-pdf` 是一个全面的 PDF 处理工具库，提供 PDF 文档的创建、读取、合并、拆分、表单填充、数字签名、内容提取等功能。采用流畅 Builder API 设计，零外部依赖，纯 JDK 25 实现。

**核心特性：**
- 文档创建：`DocumentBuilder` + `PageBuilder` 流畅链式 API
- 文档读取：支持文件、流、字节数组，支持密码保护文档
- 内容元素：文本 (`PdfText`)、段落 (`PdfParagraph`)、图像 (`PdfImage`)、表格 (`PdfTable`)、线条 (`PdfLine`)、矩形 (`PdfRectangle`)、椭圆 (`PdfEllipse`)
- 字体支持：14 种标准字体 (`StandardFont`) + 自定义嵌入字体 (`EmbeddedFont`)
- 颜色系统：RGB / RGBA / 灰度 / Hex 颜色表示
- 文档操作：合并 (`PdfMerger`)、拆分 (`PdfSplitter`)、内容提取 (`PdfExtractor`)
- 表单操作：表单填充、扁平化、字段提取，支持文本框、复选框、单选、下拉、列表、签名字段
- 数字签名：PKCS#7 签名 + 时间戳支持 + 签名验证 + 签名外观定制
- 元数据管理：标题、作者、关键词等完整元数据
- 密封类层次：`PdfElement` sealed interface、`FormField` sealed interface、`PdfFont` sealed interface

## 3. 核心 API

### 3.1 OpenPdf

> PDF 工具入口类，提供所有 PDF 操作的静态工厂方法，一站式访问文档创建、读取、合并、拆分、表单、签名、提取等功能。

**主要方法：**

| 方法 | 描述 |
|------|------|
| `static DocumentBuilder create()` | 创建 PDF 文档构建器 |
| `static DocumentBuilder create(PageSize pageSize)` | 创建指定页面大小的文档构建器 |
| `static PdfDocument open(Path path)` | 打开 PDF 文件 |
| `static PdfDocument open(InputStream inputStream)` | 从输入流打开 PDF |
| `static PdfDocument open(byte[] bytes)` | 从字节数组打开 PDF |
| `static PdfDocument open(Path path, String password)` | 打开受密码保护的 PDF |
| `static PdfMerger merger()` | 创建 PDF 合并器 |
| `static void merge(List<Path> sources, Path target)` | 合并多个 PDF 文件 |
| `static PdfDocument merge(List<PdfDocument> documents)` | 合并多个 PDF 文档 |
| `static PdfSplitter splitter()` | 创建 PDF 拆分器 |
| `static List<PdfDocument> split(Path source, String... ranges)` | 按页面范围拆分 |
| `static List<PdfDocument> splitToPages(Path source)` | 拆分为单页文档 |
| `static PdfDocument fillForm(Path source, Map<String, String> fields)` | 填充表单 |
| `static PdfDocument fillAndFlatten(Path source, Map<String, String> fields)` | 填充并扁平化表单 |
| `static Map<String, String> extractFormFields(Path source)` | 提取表单字段 |
| `static PdfSigner signer()` | 创建签名器 |
| `static PdfDocument sign(Path source, Path keyStore, char[] password, String alias)` | 签名 PDF |
| `static List<SignatureInfo> verifySignatures(Path source)` | 验证签名 |
| `static PdfExtractor extractor()` | 创建内容提取器 |
| `static String extractText(Path source)` | 提取全部文本 |
| `static String extractText(Path source, int... pageNumbers)` | 提取指定页文本 |
| `static List<byte[]> extractImages(Path source)` | 提取所有图像 |
| `static int getPageCount(Path source)` | 获取页数 |
| `static Metadata getMetadata(Path source)` | 获取元数据 |
| `static boolean isEncrypted(Path source)` | 是否加密 |
| `static boolean hasForm(Path source)` | 是否有表单 |
| `static boolean isSigned(Path source)` | 是否已签名 |

**示例：**

```java
// 创建 PDF
OpenPdf.create()
    .title("报告")
    .author("系统")
    .addPage()
        .text("标题", 100, 700)
        .paragraph(PdfParagraph.of("正文内容...", 50, 650, 500))
    .endPage()
    .save(Path.of("report.pdf"));

// 合并 PDF
OpenPdf.merge(List.of(Path.of("a.pdf"), Path.of("b.pdf")), Path.of("merged.pdf"));

// 提取文本
String text = OpenPdf.extractText(Path.of("document.pdf"));

// 签名
OpenPdf.sign(
    Path.of("contract.pdf"),
    Path.of("keystore.p12"),
    "password".toCharArray(),
    "alias"
);
```

### 3.2 PdfDocument

> PDF 文档接口，表示一个 PDF 文档，实现 `AutoCloseable`。支持页面管理、元数据、表单、保存、加密等操作。

**主要方法：**

| 方法 | 描述 |
|------|------|
| `int getPageCount()` | 获取总页数 |
| `PdfPage getPage(int pageNumber)` | 获取指定页面（1-based） |
| `List<PdfPage> getPages()` | 获取所有页面 |
| `PdfPage addPage()` | 添加空白页 |
| `PdfPage addPage(PageSize pageSize)` | 添加指定大小页面 |
| `void insertPage(int pageNumber, PdfPage page)` | 在指定位置插入页面 |
| `void removePage(int pageNumber)` | 删除页面 |
| `Metadata getMetadata()` | 获取元数据 |
| `void setMetadata(Metadata metadata)` | 设置元数据 |
| `boolean hasForm()` | 是否有表单 |
| `PdfForm getForm()` | 获取表单 |
| `void save(Path path)` | 保存到文件 |
| `void save(OutputStream outputStream)` | 保存到输出流 |
| `byte[] toBytes()` | 转字节数组 |
| `boolean isEncrypted()` | 是否加密 |
| `void setPassword(String userPassword, String ownerPassword)` | 设置密码保护 |
| `void close()` | 关闭文档 |

**示例：**

```java
try (PdfDocument doc = OpenPdf.open(Path.of("input.pdf"))) {
    int pages = doc.getPageCount();
    PdfPage page = doc.getPage(1);
    String text = page.extractText();
    Metadata meta = doc.getMetadata();
    doc.save(Path.of("output.pdf"));
}
```

### 3.4 DocumentBuilder

> PDF 文档构建器，流畅 API 创建 PDF 文档，设置元数据、页面大小、边距、字体、加密，然后添加页面。

**主要方法：**

| 方法 | 描述 |
|------|------|
| `static DocumentBuilder create()` | 创建构建器 |
| `static DocumentBuilder create(PageSize pageSize)` | 创建（指定页面大小） |
| `DocumentBuilder title(String title)` | 设置标题 |
| `DocumentBuilder author(String author)` | 设置作者 |
| `DocumentBuilder subject(String subject)` | 设置主题 |
| `DocumentBuilder keywords(String... keywords)` | 设置关键词 |
| `DocumentBuilder creator(String creator)` | 设置创建者 |
| `DocumentBuilder pageSize(PageSize pageSize)` | 设置默认页面大小 |
| `DocumentBuilder orientation(Orientation orientation)` | 设置默认方向 |
| `DocumentBuilder margins(float top, float right, float bottom, float left)` | 设置边距 |
| `DocumentBuilder margins(float margin)` | 设置统一边距 |
| `DocumentBuilder defaultFont(PdfFont font)` | 设置默认字体 |
| `DocumentBuilder embedFont(Path fontPath, String fontName)` | 嵌入自定义字体 |
| `PageBuilder addPage()` | 添加页面并进入页面构建 |
| `PageBuilder addPage(PageSize pageSize)` | 添加指定大小的页面 |
| `PageBuilder addPage(PageSize pageSize, Orientation orientation)` | 添加页面（大小+方向） |
| `DocumentBuilder encrypt(String userPassword, String ownerPassword)` | 设置加密 |
| `DocumentBuilder permissions(boolean allowPrinting, boolean allowCopying, boolean allowEditing, boolean allowAnnotations)` | 设置权限 |
| `PdfDocument build()` | 构建 PdfDocument |
| `void save(Path path)` | 保存到文件 |
| `void save(OutputStream outputStream)` | 保存到输出流 |
| `byte[] toBytes()` | 转字节数组 |

**示例：**

```java
DocumentBuilder.create(PageSize.A4)
    .title("月度报告")
    .author("财务部")
    .keywords("月报", "2026", "财务")
    .margins(50, 40, 50, 40)
    .defaultFont(StandardFont.HELVETICA)
    .addPage()
        .text("月度财务报告", 200, 750, StandardFont.HELVETICA_BOLD, 24)
        .paragraph(PdfParagraph.of("报告正文...", 50, 680, 500))
        .table(PdfTable.of(3)
            .header("项目", "预算", "实际")
            .row("人力", "100万", "95万")
            .row("设备", "50万", "48万"))
    .endPage()
    .save(Path.of("report.pdf"));
```
