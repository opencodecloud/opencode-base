# DB 组件方案

## 1. 组件概述

### 1.1 设计目标

`opencode-base-db` 模块提供轻量级数据库访问工具，简化 JDBC 操作。

**核心特性：**
- 流畅的 SQL 构建器（Select/Insert/Update/Delete）
- 自动资源管理（Connection/Statement/ResultSet）
- 轻量级 ORM（Bean/Record/Map 映射 + 注解支持）
- 事务管理（编程式事务、传播行为）
- 内置连接池（SimpleConnectionPool）
- 多数据库方言（MySQL/PostgreSQL/Oracle）
- SQL 执行监听与审计日志
- 敏感数据脱敏
- 异常自动翻译（SQLException -> 语义化异常）

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| 轻量级 | 非完整 ORM，是 JDBC 增强工具 |
| 类型安全 | 泛型映射 + 编译期检查 |
| 简洁 API | 链式调用，减少样板代码 |
| 透明性 | 可随时使用原生 SQL |
| 高性能 | 最小化反射开销，支持批量操作 |

## 3. 核心 API

### 3.1 OpenDb（数据库门面）

`OpenDb` 是数据库操作的统一入口，实现 `AutoCloseable`，支持 try-with-resources。

#### 工厂方法

| 方法 | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| `of(DataSource)` | DataSource | `OpenDb` | 从 DataSource 创建 |
| `of(ConnectionPool)` | ConnectionPool | `OpenDb` | 从连接池创建 |
| `of(String, String, String)` | url, username, password | `OpenDb` | 从 JDBC URL 创建（自动创建连接池） |
| `of(String, String, String, PoolConfig)` | url, username, password, config | `OpenDb` | 使用自定义池配置创建 |

#### SQL 构建器方法

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `select()` | `Select` | 创建 SELECT 构建器 |
| `insert()` | `Insert` | 创建 INSERT 构建器 |
| `update()` | `Update` | 创建 UPDATE 构建器 |
| `delete()` | `Delete` | 创建 DELETE 构建器 |

#### 查询方法

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `query(String sql, RowMapper<T>, Object...)` | `List<T>` | 使用自定义映射器查询 |
| `queryList(String sql, Class<T>, Object...)` | `List<T>` | 查询类型化列表 |
| `queryOne(String sql, Class<T>, Object...)` | `Optional<T>` | 查询单个对象 |
| `queryMaps(String sql, Object...)` | `List<Map<String, Object>>` | 查询为 Map 列表 |
| `queryValue(String sql, Class<T>, Object...)` | `T` | 查询单个值 |

#### 执行方法

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `execute(String sql, Object...)` | `int` | 执行更新 SQL，返回受影响行数 |
| `batchExecute(String sql, List<Object[]>)` | `int[]` | 批量执行 |

#### 实体操作方法

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `save(Object entity)` | `Object` | 插入实体，返回生成的键 |
| `modify(Object entity)` | `int` | 更新实体，返回受影响行数 |
| `remove(Object entity)` | `int` | 删除实体 |
| `removeById(Class<T>, Object id)` | `int` | 按 ID 删除 |
| `findById(Class<T>, Object id)` | `Optional<T>` | 按 ID 查找 |

#### 事务方法

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `tx(Runnable)` | `void` | 在事务中执行（无返回值） |
| `tx(Supplier<T>)` | `T` | 在事务中执行（有返回值） |
| `txManager()` | `TransactionManager` | 获取事务管理器（高级用法） |

#### 访问器方法

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `template()` | `JdbcTemplate` | 获取底层 JdbcTemplate |
| `dataSource()` | `DataSource` | 获取数据源 |
| `dialect()` | `Dialect` | 获取数据库方言 |

**使用示例：**

```java
// 创建 OpenDb
try (OpenDb db = OpenDb.of("jdbc:mysql://localhost/test", "user", "pass")) {

    // 原生 SQL 查询
    List<User> users = db.queryList(
        "SELECT * FROM users WHERE status = ?", User.class, "active");

    Optional<User> user = db.queryOne(
        "SELECT * FROM users WHERE id = ?", User.class, 1L);

    long count = db.queryValue(
        "SELECT COUNT(*) FROM users", Long.class);

    // SQL 构建器查询
    List<User> result = db.select()
        .from("users")
        .where(Condition.eq("status", "active"))
        .orderBy("created_at", false)
        .limit(10)
        .list(User.class);

    // 执行更新
    int affected = db.execute(
        "UPDATE users SET name = ? WHERE id = ?", "Zhang San", 1L);

    // 实体操作
    User newUser = new User();
    newUser.setName("Zhang San");
    db.save(newUser);

    Optional<User> found = db.findById(User.class, 1L);

    // 事务
    db.tx(() -> {
        db.execute("UPDATE accounts SET balance = balance - ? WHERE id = ?", 100, 1L);
        db.execute("UPDATE accounts SET balance = balance + ? WHERE id = ?", 100, 2L);
    });
}
```

### 3.2 JdbcTemplate（JDBC 模板）

`JdbcTemplate` 是核心执行类，封装了 JDBC 操作的所有细节。

```java
public class JdbcTemplate {
    public JdbcTemplate(DataSource dataSource);
    public JdbcTemplate(DataSource dataSource, Dialect dialect);
}
```

#### 查询方法

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `queryOne(String sql, Class<T>, Object...)` | `Optional<T>` | 查询单个对象（结果超过1个抛异常） |
| `queryList(String sql, Class<T>, Object...)` | `List<T>` | 查询列表（自动选择映射器） |
| `query(String sql, RowMapper<T>, Object...)` | `List<T>` | 使用自定义映射器查询 |
| `queryValue(String sql, Class<T>, Object...)` | `T` | 查询单个值 |
| `queryMaps(String sql, Object...)` | `List<Map<String, Object>>` | 查询为 Map 列表 |

#### 更新方法

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `update(String sql, Object...)` | `int` | 执行更新 |
| `batchUpdate(String sql, List<Object[]>)` | `int[]` | 批量更新 |
| `insert(String sql, Class<T> keyType, Object...)` | `T` | 插入并返回生成的主键 |

#### 实体方法

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `findById(Class<T>, Object id)` | `Optional<T>` | 按 ID 查找实体 |
| `findById(Class<T>, Object id, EntityMeta)` | `Optional<T>` | 按 ID 查找（指定元数据） |
| `save(T entity)` | `T` | 保存实体（自动设置生成的 ID） |
| `save(T entity, EntityMeta)` | `T` | 保存实体（指定元数据） |
| `updateEntity(T entity)` | `int` | 更新实体 |
| `updateEntity(T entity, EntityMeta)` | `int` | 更新实体（指定元数据） |
| `delete(T entity)` | `int` | 删除实体 |
| `delete(T entity, EntityMeta)` | `int` | 删除实体（指定元数据） |
| `deleteById(Class<T>, Object id)` | `int` | 按 ID 删除 |

#### 事务与监听

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `transaction(Supplier<T>)` | `T` | 在事务中执行 |
| `addListener(SqlExecutionListener)` | `void` | 添加 SQL 执行监听器 |
| `removeListener(SqlExecutionListener)` | `void` | 移除监听器 |
| `getDataSource()` | `DataSource` | 获取数据源 |
| `getDialect()` | `Dialect` | 获取方言 |

**使用示例：**

```java
JdbcTemplate template = new JdbcTemplate(dataSource);

// 查询
List<User> users = template.queryList(
    "SELECT * FROM users WHERE age > ?", User.class, 18);

// 插入并获取主键
Long newId = template.insert(
    "INSERT INTO users (name, email) VALUES (?, ?)",
    Long.class, "Zhang San", "zs@example.com");

// 批量更新
List<Object[]> batchArgs = List.of(
    new Object[]{"User1", "u1@test.com"},
    new Object[]{"User2", "u2@test.com"}
);
int[] results = template.batchUpdate(
    "INSERT INTO users (name, email) VALUES (?, ?)", batchArgs);

// 事务
template.transaction(() -> {
    template.update("UPDATE accounts SET balance = balance - 100 WHERE id = ?", 1L);
    template.update("UPDATE accounts SET balance = balance + 100 WHERE id = ?", 2L);
    return null;
});
```

```java
List<Map<String, Object>> result = db.select()
    .columns("u.name", "COUNT(o.id) as order_count")

    .where(Condition.ge("u.age", 18))
    .groupBy("u.id")
    .having(Condition.gt("COUNT(o.id)", 5))
    .orderBy("order_count", false)
    .limit(10)
    .listMaps();

// 分页
List<User> page = db.select()
    .from("users")
    .where(Condition.eq("status", "active"))
    .orderBy("id")
    .limit(20)
    .offset(40)
    .list(User.class);

// 计数和存在检查
long total = db.select().from("users").count();
boolean exists = db.select().from("users")
    .where("email", "test@example.com").exists();
```

### 4.5 Condition（条件构建器）

```java
public class Condition {
    // 比较操作
    public static Condition eq(String column, Object value);        // =
    public static Condition ne(String column, Object value);        // !=
    public static Condition gt(String column, Object value);        // >
    public static Condition ge(String column, Object value);        // >=
    public static Condition lt(String column, Object value);        // <
    public static Condition le(String column, Object value);        // <=

    // 模式匹配
    public static Condition like(String column, String pattern);    // LIKE
    public static Condition contains(String column, String value);  // LIKE '%value%'
    public static Condition startsWith(String column, String value);// LIKE 'value%'
    public static Condition endsWith(String column, String value);  // LIKE '%value'

    // 范围
    public static Condition in(String column, Object... values);    // IN (?, ?, ...)
    public static Condition notIn(String column, Object... values); // NOT IN
    public static Condition between(String column, Object from, Object to); // BETWEEN

    // NULL 检查
    public static Condition isNull(String column);
    public static Condition isNotNull(String column);

    // 特殊
    public static Condition raw(String sql, Object... params);   // 原始 SQL 条件
    public static Condition alwaysTrue();                        // 恒为 true
    public static Condition alwaysFalse();                       // 恒为 false

    // 组合
    public Condition and(Condition other);
    public Condition or(Condition other);
    public Condition not();

    // 输出
    public String toSql();
    public List<Object> getParams();
}
```

**使用示例：**

```java
// 简单条件
Condition c = Condition.eq("status", "active");

// 组合条件
Condition c = Condition.eq("status", "active")
    .and(Condition.ge("age", 18))
    .and(Condition.isNotNull("email"));

// OR 条件
Condition c = Condition.eq("role", "admin")
    .or(Condition.eq("role", "superadmin"));

// IN 查询
Condition c = Condition.in("status", "active", "pending", "review");

// BETWEEN
Condition c = Condition.between("created_at",
    LocalDate.of(2024, 1, 1), LocalDate.of(2024, 12, 31));

// 模糊匹配
Condition c = Condition.contains("name", "zhang"); // name LIKE '%zhang%'

// 原始 SQL
Condition c = Condition.raw("YEAR(created_at) = ?", 2024);
```

```java

            .set("product_id", item.getProductId())
            .execute();
    }

    // 执行操作...
    tx.commit();
} catch (Exception e) {
    tx.rollback();
    throw e;
} finally {
    tx.close();
}

// 指定传播行为
txManager.execute(Propagation.REQUIRES_NEW, () -> {
    // 独立事务
```
